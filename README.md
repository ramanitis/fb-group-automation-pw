# FB Group Automation (TypeScript)

Small TypeScript/Node.js project scaffold. Currently logs a greeting from `src/index.ts`; extend it with your automation logic.

## Prerequisites
- Node.js 18+ and npm

## Setup
```bash
npm install
```

## Development
- Run directly with ts-node: `npm run dev`

## Build & Run
```bash
npm run build
npm start
```

## Project Structure
- `src/index.ts` - entry point
- `dist/` - compiled output after `npm run build`
